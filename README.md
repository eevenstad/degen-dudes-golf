# Degen Dudes Golf App

Mobile-first golf scoring app for thedegendudes.com.

Built with Next.js 15, Supabase, Tailwind CSS, and deployed on Vercel.

## Setup
See the setup guide at erice.me/briefs/degen-dudes-setup-guide

## Development
```bash
npm run dev
```

## Tech Stack
- Next.js 15 (App Router, TypeScript)
- Supabase (PostgreSQL, Auth, Real-time)
- Tailwind CSS + shadcn/ui
- Vercel (hosting)
