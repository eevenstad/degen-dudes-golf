### 4f: Singles Stroke Play (`singles_stroke`)

Same player matchup and scores as 4d (Eric vs Ben), but winner determined by total PH-net across 18 holes, not hole-by-hole.

**How it works in the app:** No points awarded on individual holes. After all 18 holes are complete, the player with the lower total PH-net score wins the match (1 point). If tied, 0.5 each.

**Using 4d scores:** Sum Eric's PH-net column and Ben's PH-net column from section 4d above. The lower total wins.

**Key code path:** In `scores.ts` → `updateMatchPoints()`, singles_stroke only assigns points when `holesCompleted === 18`. Before that, both team_a_points and team_b_points stay at 0. This is an important test — verify no points appear mid-round.

### Scoring Formulas
```
CH = ROUND(HI × (Slope / 113) + (Rating - Par))
PH = CH - min(group CH)
Strokes: CH≥36 → rank≤(CH-36)?3:2; CH≥18 → rank≤(CH-18)?2:1; else rank≤CH?1:0
Net = min(gross - strokes, par + strokes + NET_MAX_OVER_PAR)
```
